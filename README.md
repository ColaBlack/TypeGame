# TypeGame

你，是不是用惯了手机，打字速度很慢啊？
你想提高打字速度吗？
看看这个项目吧，一个双人PVP打字小游戏！


### 技术栈
- easyX图形绘制
- httplib进行网络交互，搭建客户端服务器
